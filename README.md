# 人流アノマリー要因分析アプリ

日時と緯度・経度を入力すると、**Claude（Anthropic）がWeb検索を横断して人流アノマリー（異常な増加）の原因を分析**し、Markdown形式のレポートを生成するモバイルWebアプリ（PWA）です。

位置情報アナリスト／データサイエンティストの視点で、
「どこで（Where）」「なぜ（Why）」を深掘りし、**「データで言える事実」と「Web情報からの推定」を区別**したレポートを作成します。

## 特長

- 📱 **PWA** — スマホのブラウザで開き、ホーム画面に追加すればネイティブアプリのように使えます（インストール不要）
- 🔎 **Web横断調査** — Claude の `web_search` / `web_fetch` サーバーツールで、イベント・ニュース・災害/気象・交通・SNSなどをリアルタイムに調査
- 📍 **現在地取得** — ワンタップで端末のGPSから緯度経度を入力
- 🔐 **APIキーは端末内のみ** — 入力したAnthropic APIキーはブラウザの `localStorage` にのみ保存。外部サーバーには送信されず、Claude API へのリクエストにのみ使用されます（バックエンド不要）
- ⚡ **ストリーミング表示** — 調査の進捗（検索クエリ・取得ページ）をリアルタイム表示し、レポートを逐次描画
- 📋 レポートの **コピー / Markdownダウンロード** に対応
- 🌗 ライト/ダークモード自動対応

## 使い方

1. アプリを開く（下記「公開方法」参照）
2. 上部「**API設定**」を開き、Anthropic APIキー（`sk-ant-...`）を入力
   - キーは [console.anthropic.com](https://console.anthropic.com) で取得できます
3. **日時**・**緯度**・**経度** を入力（任意で観測状況／周辺施設／周辺地形も）
4. 「**レポートを生成**」をタップ
5. 調査完了後、レポートが表示されます（コピー／保存可能）

## 使用モデル

- 既定: **Claude Opus 4.8**（`claude-opus-4-8`） — 要因の深掘りとWeb調査の精度が最も高い
- 切替可: **Claude Sonnet 5**（`claude-sonnet-5`） — 高速・低コスト

Web検索・Webフェッチのサーバーツール（`web_search_20260209` / `web_fetch_20260209`）を利用します。

## 公開方法（いずれか）

これは**ビルド不要の静的サイト**です。以下のいずれかで配信できます。

- **ローカル確認**: このディレクトリで `python3 -m http.server 8080` を実行し、スマホから `http://<PCのIP>:8080/` を開く
- **GitHub Pages**: リポジトリの Settings → Pages で本ブランチ／ルートを公開
- 任意の静的ホスティング（Netlify / Vercel / Cloudflare Pages / S3 など）にファイル一式をアップロード

> ⚠️ PWA（Service Worker・ホーム画面追加）は **HTTPS**（または `localhost`）でのみ有効です。GitHub Pages等はHTTPSで配信されるため問題ありません。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（UI・API呼び出し・Markdownレンダラをすべて内包） |
| `manifest.webmanifest` | PWAマニフェスト |
| `sw.js` | Service Worker（アプリシェルのオフラインキャッシュ。API通信はキャッシュしません） |
| `icon.svg` / `icon-192.png` / `icon-512.png` | アプリアイコン |

外部ライブラリ・CDNへの依存はありません（Markdownレンダラも内蔵）。

## 注意事項

- レポートには **Web情報に基づく推定（仮説）** が含まれます。重要な判断の前に一次情報でご確認ください。
- APIの利用にはAnthropicの利用料金が発生します。
- 位置情報・入力内容・検索クエリは、レポート生成のためAnthropic APIに送信されます。
